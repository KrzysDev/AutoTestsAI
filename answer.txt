Na podstawie przesłanego planu działania i wymaganej struktury JSON, stworzyłem dwa grupy pytań testowych dla uczniów na poziomie B2 w języku angielskim. Test skupia się na różnych aspektach czasów przeszłych: Simple Past Tense, Past Continuous Tense i Past Perfect Tense.

Oto przykładowy JSON z dwoma grupami pytań:

```json
{
  "$defs": {
    "Group": {
      "properties": {
        "questions": {
          "items": {
            "$ref": "#/$defs/Question"
          },
          "title": "Questions",
          "type": "array"
        },
        "answers": {
          "items": {
            "type": "string"
          },
          "title": "Answers",
          "type": "array"
        }
      },
      "required": [
        "questions",
        "answers"
      ],
      "title": "Group",
      "type": "object"
    },
    "Question": {
      "properties": {
        "text": {
          "title": "Text",
          "type": "string"
        },
        "type": {
          "enum": [
            "multiple_choice",
            "open_ended"
          ],
          "title": "Type",
          "type": "string"
        },
        "correct_answer": {
          "title": "Correct Answer",
          "type": "string"
        }
      },
      "required": [
        "text",
        "type",
        "correct_answer"
      ],
      "title": "Question",
      "type": "object"
    }
  },
  "properties": {
    "groups": {
      "items": {
        "$ref": "#/$defs/Group"
      },
      "title": "Groups",
      "type": "array"
    }
  },
  "required": [
    "groups"
  ],
  "title": "Test",
  "type": "object",
  "groups": [
    {
      "questions": [
        {
          "text": "What was Tom doing when his friend called him?",
          "type": "multiple_choice",
          "correct_answer": "was studying"
        },
        {
          "text": "How did Sarah feel after her presentation at work yesterday?",
          "type": "multiple_choice",
          "correct_answer": "felt nervous"
        }
      ],
      "answers": [
        "Tom was studying",
        "Sarah felt nervous"
      ]
    },
    {
      "questions": [
        {
          "text": "Complete the sentence with the correct form of 'to have': 'By the time she arrived, I _______ finished my homework.'",
          "type": "open_ended",
          "correct_answer": "had"
        },
        {
          "text": "Write a short story about your last summer holiday using past tense verbs.",
          "type": "open_ended",
          "correct_answer": ""
        }
      ],
      "answers": [
        "By the time she arrived, I had finished my homework.",
        "Your narrative here"
      ]
    }
  ]
}
```

W powyższym JSONie każda grupa zawiera dwa zadania: jedno z wielokrotnego wyboru i drugie do uzupełnienia luki lub pisania zdania. To jest tylko przykładowy model; w rzeczywistym testie będą one były bardziej szczegółowe i obszerne, zawierając więcej przykładów i różnych typów pytań.

Jeśli chcesz, aby test był bardziej szczegółowy lub miał inne struktury, daj mi znać, a dostosuję go według Twoich potrzeb.
